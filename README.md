# HopeBridge Foundation

## Task Name

AI Website Generation

## Domain

NGO / Social Impact

## Tools Used

AI-assisted coding with Codex, React, Vite, and CSS.

## Project Summary

HopeBridge Foundation is a responsive demo website for a nonprofit-style social impact organization focused on education, food assistance, women empowerment, animal welfare, environmental awareness, and skill development. The content is generic demo content and does not include fake registrations, certifications, or legal claims.

## Sections Included

- Header and responsive navigation
- Hero section with donation and volunteer calls to action
- About section with Compassion, Community, and Action value cards
- Causes cards for education, food, women empowerment, animal welfare, environment, and skills
- Services section for practical outreach programs
- Impact statistics
- Gallery / work preview section with accessible placeholder visuals
- Demo testimonials
- Contact form and contact details
- Footer with quick links, social placeholders, and demo copyright

## How to Run

```bash
npm install
npm run dev
npm run build
```

## Deployment Notes

This project is ready to deploy on Vercel as a Vite React app. Use `npm run build` as the build command and `dist` as the output directory.
